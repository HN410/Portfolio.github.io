Simulator for our home-built CPU (CLI, GUI)

Our undergraduate curriculum includes “CPU Experiment” in which students design and develop CPUs from scratch in groups. This is a simulator that behaves in the same way as our group’s CPU, which we have created. The simulator has been improved over nearly six months by providing GUI software to facilitate debugging by group members and adding various functions such as a rewind function and memory access count ranking.
Language C++ (CLI)
Java (GUI)
Lines About 5,000 (CLI)
About 2,400 (GUI)
GitHub CLI, GUI
